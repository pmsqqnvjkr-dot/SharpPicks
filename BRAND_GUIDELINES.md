# Sharp Picks Brand Guidelines

---

## Brand Identity

**Brand Name:** Sharp Picks
**Tagline:** One pick beats five.
**Subject Line Format:** "Welcome to Sharp Picks | The Edge is Discipline"
**Tone:** Calm, institutional, data-driven. No hype, no FOMO, no flashy promos.

### Brand Voice
- Measured and confident, never loud
- Data-first — let the numbers speak
- Institutional-grade language, not gambling slang
- Founder-led authenticity (Evan Cole)
- Discipline over volume, always

### Key Phrases
- "Discipline is the edge."
- "Sharp Principle" (used for pull-quote blocks)
- "The goal isn't just to win a bet; it's to build a sustainable edge."
- "Verified by data, not talk."
- "We treat it like a market."
- "Pass days aren't missed opportunities — they're proof the system is working."

## Typography

### Font Stack

| Role | Font Family | Fallback | Usage |
|------|------------|----------|-------|
| Sans (Primary) | Inter | -apple-system, BlinkMacSystemFont, sans-serif | Body text, UI elements, navigation |
| Serif (Display) | IBM Plex Serif | Georgia, serif | Headlines, pull quotes, Sharp Principles |
| Mono (Data) | JetBrains Mono | SF Mono, monospace | Statistics, odds, percentages, labels |

### Weight Scale
- **300** — Light: Subtle body text
- **400** — Regular: Standard body
- **500** — Medium: Emphasized body, sub-headers
- **600** — Semi-bold: Section headers, key metrics
- **700** — Bold: Headlines, CTAs, numbered steps
- **800/900** — Extra-bold: Hero headlines (landing page only)

### Label Style
Small uppercase labels (used for categories like "Sharp Principle", "Weekly Summary"):
```
font-family: 'Courier New', monospace
font-size: 9px
font-weight: 700
letter-spacing: 2.5px
text-transform: uppercase
color: #34D399 (green) or #4F86F7 (blue)
```

## Logo

### Versions
- **Primary (Shield):** Shield icon with chart line and bars — used in app header, emails
- **Logo (Square):** `logo.png` / `logo-1024.png` — app icon, social media
- **Horizontal:** `logo-horizontal.png` — email headers, wide contexts
- **Email-optimized:** `logo-email.png` — embedded in transactional emails

### Logo Usage
- Always on dark backgrounds (`#0A0D14`)
- Maintain clear space equal to the shield height on all sides
- Never alter colors or proportions
- Shield icon can be used standalone at small sizes (24px+)

### Shield Icon (SVG)
The inline SVG shield contains:
- Shield outline (white stroke, 1.8px)
- Three vertical bars (varying opacity: 0.3, 0.4, 0.35)
- Upward trend line with arrow endpoint

## Photography & Imagery

- No stock photos of sports/gambling
- No flashy graphics or neon effects
- Abstract data visualizations preferred
- Blurred chart overlays for gated content
- Dark, minimal, institutional aesthetic
- SVG icons preferred over raster images

## Compliance

All public-facing content must include:
> "This is entertainment only. Past results don't guarantee future performance."

---

## Don'ts

- Don't use gambling language ("lock," "guaranteed," "can't miss")
- Don't use FOMO tactics ("hurry," "limited time," "act now")
- Don't use bright/neon color schemes
- Don't use light/white backgrounds
- Don't use emojis in emails or UI
- Don't promise specific win rates in marketing copy
- Don't use exclamation marks excessively
- Don't reference specific dollar amounts won/lost in promotions
