Standard card structure (top to bottom):

  1. Left state accent bar: sage on win, rose on loss, neutral on push.
  2. Header row: state pill (Win / Loss / Push) left, date dot sport right.
  3. Body:
    • Matchup eyebrow ("Indiana Fever vs Washington Mystics", mono).
    • Pick row: serif pick name plus muted line, P&L units right-aligned.
    • Score row: "Final dot AWY 98 dot HOM 93" left, dollar P&L right (hidden when bet not tracked).
  4. Footer: CLV pill + value left, Share + View outcome actions right.

Actions in footer:

  • "Share" (secondary, muted) -> existing share infrastructure (handleShareResult).
  • "View outcome" (primary, Signal Blue) -> opens ResolutionScreen via setResolutionPick + setShowResolution.
