The two-spec system (v4.3 lock)

DELIBERATE DECISION

Wordmark and standalone signal mark use M2 medium bar weight. App icon uses light bar weight. These are different rendering contexts and need different geometry. Do not "harmonize" them.

Why the two-spec system:

  • App icon: renders at notification-badge sizes (29x29px) and tab icon sizes (60x60px) where bar widths are measured in fractional pixels. Light weight (3.8% x 34%) survives. Medium weight at those sizes turns into chunky rectangles that lose typographic identity.
  • Wordmark and standalone signal mark: render in typographic contexts where bar weight should harmonize with surrounding letter weight. Medium weight (0.12em x 1.30em) has presence without dominating. Light weight at small wordmark sizes (16-18px UI headers) collapses visually.

Wordmark exact values (v4.3)

  • Font: Inter weight 500 (Medium) — never 300, never 600
  • Letter-spacing: 0.25em
  • Case: ALL UPPERCASE
  • Bar width: 0.12em (M2 medium)
  • Bar height: 1.30em (15% overshoot above and below cap-height)
  • Bar gap: 0.20em
  • Margin around signal cluster: 0 0.5em 0 0.22em (asymmetric, optical correction)
  • Bar border-radius: 999px (full pill)
  • Color on dark: Sharp White #E8EAED
  • Color on light: Sharp Navy #111827
  • Min render width: 120px (below this, switch to standalone signal mark)

Standalone signal mark exact values (v4.3)

  • Bar width: 0.15em (slightly heavier than wordmark inline bars to compensate for standalone context)
  • Bar height: 1em
  • Bar gap: 0.22em
  • Underline width: 0.5em · height: 0.07em · margin-top: 0.18em
  • Underline color: Edge Green #5A9E72 (always present, non-negotiable)

App icon exact values (v4.3, light weight intentionally)

  • Background: Sharp Navy #111827 (the only place this color appears in the system)
  • Bars: Sharp White, 3.8% width × 34% height of icon
  • Bar gap: ~3.9% of icon width
  • Green accent: 15% width × 1.2% height, Edge Green, centered below bars
  • Below 40x40px: omit green accent

Tagline lockup exact values

  • Text: "ONE PICK BEATS FIVE" (uppercase) or rotates through other Sharp Principles
  • Font: Inter weight 400
  • Letter-spacing: 0.35em
  • Color: Edge Green #5A9E72
  • Size: 35% of wordmark font-size
  • Position: centered below wordmark, gap of 1.4em (relative to wordmark size)

Construction approaches

Canonical (modern browsers): uses display: inline-flex with align-items: center. Cleanest code. Implementation shown in the CSS at the top of this file.

Fallback (render-anywhere): uses absolute positioning to force cap-line-to-baseline placement. Required when targeting PDF generators (wkhtmltoimage, wkhtmltopdf), server-side image renderers, or older WebKit. Full implementation in BRAND_SPEC.md v4.3.
