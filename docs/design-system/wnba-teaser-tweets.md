## What changes in the broader strategy

The "calibration phase, signals live" framing actually opens up tweet types that weren't possible under the original "no signals all season" plan:

1. **Signal-fired alerts** — when a signal clears the threshold, you can post about it in real time. The tweet still carries the calibration phase tag, so expectations are set, but you get to post about active opportunities
2. **Outcome posts** — signals that win can be posted as wins. Signals that miss get posted as misses, with closing line audit. You build a public ledger from week one
3. **Confidence calibration narrative** — every week or two, you can post about how the model's confidence is trending. "Two weeks in, the model has overweighted home court by ~0.4 points. Adjusting." That's the kind of transparency no other betting account offers

This is a stronger story than the original plan. Build the launch around it.
